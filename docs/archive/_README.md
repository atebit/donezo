# docs/archive

This directory holds point-in-time pre-rebuild artifacts that are no longer maintained.
They exist as historical record only — do not update them.
For legacy-code archaeology (the original CRA + Express + MongoDB application), see git commit `a5d47c2`.

## Subtrees

- **`audit/`** — 12-file inventory of the legacy codebase, feature gaps, and the migrate-now decision memo that launched the rebuild.
- **`pre-planning/`** — 6-file predecessor to `docs/conversion-plan/`: architecture notes, feature inventory matrix, gap analysis, roadmap, and Google login planning.
- **`conversion-refinements/`** — 1 stray refinement doc (`auth-google-only.md`) from early epic 03 planning; superseded by the shipped auth implementation.
